Van moodboard tot prototype

Ik ben begonnen met een moodboard. Dit heb ik gedaan door de brainstormen rond mijn idee, ik wou werken met zombies, het leger en vliegtuigen. Online heb ik inspiratie opgedaan rond deze onderwerpen en ik heb deze afbeeldingen verzameld op één canvas. Uit deze foto's ben ik kleuren gaan halen, waarmee ik mijn kleurenpallet heb samengesteld.

JavaScript

Om mijn website te verfijnen, heb ik enkele dingen in JavaScript toegevoegd.

Hieronder zie je een timer die imiteert dat het spel aan het downloaden is gedurende 10 seconden. Na afloop komt er op het scherm te staan dat de download voltooid is.

Next picture

Op mijn website heb ik ook gebruik gemaakt van een plug-in, namelijk Simple Parallax. Deze plug-in zorgt voor parallax animaties bij afbeeldingen. Ik heb deze gebruikt op de about pagina. Als je scrollt, zie je de foto's bewegen.


Als laatste heb ik er voor gezorgd dat al mijn foto's gecomprimeerd zijn zodat ze zo klein mogelijk zijn. En naast de vereisten, heb ik mij ook wat geamuseerd met mijn website door nieuwe dingen uit te proberen. Zo heb ik een custom cursor toegevoegd wanneer je bovenaan op de links klikt.
